# Advent Of Code

Solutions to the [Advent of Code](https://adventofcode.com/) programming puzzles.

## Python

To run the test suite use the `setup.py` test runner, like so:

```
$ cd python
$ python3.7 -m venv venv
$ . venv/bin/activate
(venv) $ python3 setup.py test
```
